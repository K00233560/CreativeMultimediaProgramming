LIT Higher Diploma Creative Multimedia Programming

As part of the evolving web technology module in the Creative Multimedia Programming Higher Diploma at Limerick Institute of Technology, we have to create a website to demonstrate how search engine optimisation works. The Creative Multimedia Course is a one year programme ran by springboard available to postgraduates who would like make a transition into the IT sector.

The modules of the course include:
Semester 1:

  • Web Development
  • Web Programming
  • Interaction Design
  • Digital Video
  • 3D Graphics
  • Human Cognition and Communications
  • Databases
Semester 2:
  • Advanced Web Programming
  • Evolving Web Technology
  • Interactive Interface Design
  • Project
  • Work Placement
