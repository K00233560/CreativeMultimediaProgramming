About this Website


LIT Higher Diploma Creative Multimedia Programming 2018

As part of the evolving web technology module for the H. Dip Creative Multimedia Programming Course at Limerick Institute of Technology, we have to create a website to demonstrate how search engine optimisation works. The Creative Multimedia Course is a one year programme ran by springboard available to postgraduates who would like make a transition into the IT sector.
The course helps non-IT postgraduates develop technical and creative skills to prepare them for a career in the multimedia technology industry - an industry which there is a growing skills shortage in Ireland

Level 8 Higher Diploma in Creative Multimedia Programming at LIT : Course Details

Duration: 1 Year Full-Time

Learners will engage in active learning through:

  • Attendance on-site at LIT Moylish Campus
  • Lectures
  • Tutorials & Practicals

The following modules will be covered:
For more information on each please see Modules

  1. Semester 1
    • Web Development
    • Web Programming
    • 3D Graphics
    • Interaction Multimedia Design
    • Multimedia Digital Video
    • Multimedia Database Systems
    • Human Cognition and Communication

  2. Semester 2
    • Advanced Web Programming
    • Evolvinging Web Technologies
    • Interactive Interface Design
    • Interaction Multimedia Design
    • Multimedia Project

  3. Semester 3
    • 3 month Work Placement
      (During summer months)

Entry Requirements

Applicants must satisfy the minimum entry requirement of a Level 8 degree, or equivalent, in a non-cognate area.

Application Procedure

Applicants must apply at www.springboardcourses.ie
